## Project Overview
This project analyzes SpaceX Falcon 9 launch data to predict the success of first-stage landings using data science and machine learning techniques.

## Objectives
- Collect SpaceX launch data using REST API and web scraping
- Perform data wrangling and exploratory data analysis
- Visualize launch patterns using maps and dashboards
- Build and evaluate classification models to predict launch success
